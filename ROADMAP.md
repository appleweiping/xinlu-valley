# 新路谷物语 Newroad Valley — 建设日志与路线图

升级纪律（2026-06-11 确立）：**按大版本推进。做完一个大里程碑，立刻规划下一个；
每次"升级"必须是成体系的版本跃迁（多个子系统、可感知的内容/玩法跨越），
严禁 toy 式小迭代。** 用户说"升级" = 自动执行下一个排队中的大版本。

每个版本的完成标准：本地验证通过 → 构建零错 → Vercel 部署 → GitHub 推送 →
**密钥泄漏扫描通过后发布 GitHub Release（tag vN.x.x，让更多人看到）** →
本文件与落地页建设日志更新 → agentmemory 登记。

## ✅ v4 · 室内与活水 Interiors & Living Data（2026-06）

从"能逛的镇子"跃迁为"能住、能干活的镇子"：

- **六个可进入室内**（数据驱动单场景引擎）：记忆图书馆书架长廊、技能工坊
  锻造间、市政厅状态大厅、农舍卧室（床=存档+睡到次日）、温室育苗间、
  研究大厅看板墙。暖/冷两种石墙、门口转场、室内交互物直开数据面板。
  自绘像素道具补缺：彩书脊书架、服务状态大屏、锻炉、看板（tools/props.py）。
- **作物=真实任务**：农场与 workspace/tasks.json 双向绑定——种植=建任务、
  浇水=推进、收获=完成写回（演示模式走 localStorage 模拟）。LPC 作物
  多品种按任务名哈希分配。
- **NPC 日程 + A\* 寻路**：上午/下午进岗位建筑（玩家进楼能遇到驻点居民）、
  午间广场聚集、夜晚回旅店；对话首行报告当前活动。
- **对话桥**：联机模式对话框内可继续提问——有密钥走 OpenAI 兼容渠道按
  人设回答；无密钥用真实数据模板（Sonnet 引用真实记忆标题等）。
- **音频**：全 CC0（Apple Cider BGM + Kenney/OGA 音效），后台延迟加载
  不阻塞进镇；HUD 静音钮。
- **存档**：位置/天数/时钟/收获/演示作物 localStorage；门口转场自动存档。
- **页内 E2E 冒烟器**（?v4test=1）：逐步把验证证据（JSONL+画布截图）
  POST 落盘到 workspace/，浏览器工具再不稳也能验收。
