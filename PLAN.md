# Agent Town Godot Rebuild Plan

Updated: 2026-05-30

## Objective

Rebuild Agent Town as a top-tier open-source Godot game that turns the local computer into a playable work world. The player should be able to do local office work, file management, software development, multi-agent coordination, agentmemory use, knowledge-base browsing, and research workflows while playing. Gameplay should move toward a Rock Kingdom-like map/NPC/task/collection structure while preserving one consistent AI-town art direction.

## Current Verification Criteria

- Godot project exists and opens from `godot/project.godot`.
- Project uses Godot 4.6.3 stable and GDScript.
- First playable slice includes click-to-move, resident agents, building hotspots, and local-data inspection through the existing backend.
- Local startup scripts open only this project's backend and Godot windows.
- Existing external experiments and unrelated processes are not stopped or modified.
